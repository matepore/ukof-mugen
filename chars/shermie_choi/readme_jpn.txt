●AI設定
shermie_cmd.txtを開き[State -1, AI config]の最後のvar(51)から1つ選んでください。いつもの私の作る強さです。対人戦目的の人は投げなど反応が悪い(51) =1を推奨します。

●脱衣KO演出の選択
敗北時(2本先取なら2敗目で)にKOされると特殊なやられになります。この演出にしたい人は、shermie_cmd.txtを開き、0の前に;をつけて、1の前の;を削除してください。
;Stripping KO ON(1)/OFF(0)
[State -1, setting]
type = varset
trigger1 = 1
v = 59
;value = 0
value = 1

特殊KOの揺れは拘ったかな。大きい胸はドット打ちで遊べて面白いですね。
