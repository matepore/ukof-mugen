\\\\\\\\ AIの使い方 \\\\\\\\

tobikageファイルとdefファイルを突っ込む

そうしてMUGEN本体のselect.defに以下を記述

Krizalid_N			    ;にゃんころ氏AI
Krizalid_N/Krizalid_N_tobikage.def　;鳶影AI

\\\\\\\\\\動画使用の場合\\\\\\\\

AI.txtを開いて

[State -3,常時AI起動]
Type = VarSet
Triggerall = roundstate != 1
Triggerall = roundstate = 2
Triggerall = roundstate != 3
Triggerall = roundstate != 4
trigger1 = !enemynear,vel x = 0
trigger2 = !enemynear,vel y = 0
trigger3 = enemynear,ctrl
V = 42
value = 0

0を1にしてください。

元に戻したいときは逆のことをしてください。

\\\\\\\AIレベル設定\\\\\\\\

AI.ｔｘｔファイルを開いて

;--------------------------
;AI
;--------------------------
[State -1,AI]
type = Varset
triggerall = !var(20)
triggerall = RoundState = [1,2]
trigger1 = command = "AIflag1"
trigger2 = command = "AIflag2"
trigger3 = command = "AIflag3"
trigger4 = command = "AIflag4"
trigger5 = command = "AIflag5"
trigger6 = command = "AIflag6"
trigger7 = command = "AIflag7"

　　　　(略)

var(20) = 11　;Lv 1~11

で調節してください
