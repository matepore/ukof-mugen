\\\\\\\ AIの使い方とモード設定 \\\\\\\\

tobikageファイルと各defファイルを突っ込む

そしてMUGENのdataファイルのselect.defに↓を記述

Kim_Kaphwan		       ;AI無し
Kim_Kaphwan/Kim_AI.def　       ;ノーマル・EXモードのAIが起動します。。
Kim_Kaphwan/Kim_AI-mix.def　　 ;MIXモードのAIが起動します。


※例えMIXのAIを使用していたとしてもプレイヤー操作で同時に両方の技は使用できません。

\\\\\\\\\\動画使用の場合\\\\\\\\

Kim_Kaphwan.cmdファイルを開いて

[State -3,常時AI起動]
Type = VarSet
Triggerall = roundstate != 1
Triggerall = roundstate = 2
Triggerall = roundstate != 3
Triggerall = roundstate != 4
trigger1 = !enemynear,vel x = 0
trigger2 = !enemynear,vel y = 0
trigger3 = enemynear,ctrl
V = 43
value = 0

0を1にしてください。

元に戻したいときは逆のことをしてください。

\\\\\\\AIレベル設定\\\\\\\\

Kim_Kaphwan.cmdファイルを開いて

[State -1, 1]
type = Varset
triggerall = !Var(59)
triggerall = RoundState = [1,2]
trigger1 = Command = "AI0"
trigger2 = Command = "AI1"
trigger3 = Command = "AI2"
trigger4 = Command = "AI3"
trigger5 = Command = "AI4"
trigger6 = Command = "AI5"
trigger7 = Command = "AI6"

　　　　(略)

var(59) = 11　;Lv 1～11

で調節してください
