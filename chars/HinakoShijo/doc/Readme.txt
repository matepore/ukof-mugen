===============================================================================
 はじめに
===============================================================================
ダウンロードしていただきありがとうございます。
このキャラは KOF2000 + 2001 + 2003 + 2002UM の要素を複合させたような四条雛子です。
初めての製作なので不具合などあるかもしれないです。

===============================================================================
 注意
===============================================================================
このキャラの無断改変、転載はOKです。
ただしこのキャラクターの画像や音声を使用する場合は自己責任でお願いします。

ファイルへの直リンクはご遠慮ください。

===============================================================================
 設定できる項目について
===============================================================================
"option.txt"というファイル内で、
「ここから」「ここまで」と囲われた部分の設定を変えることが出来ます。

またAIの時、11、12Pカラーを選ぶ事で強設定モードになります。
