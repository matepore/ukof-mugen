Orochi-EV 1.54

Critical Bug Fix + "Ture God" Mode + Now u can choose to disable strikers in Nightmare Mode
-----------------------------------------------------------------
Fix a critical bug on Mode Switch that cause entering God Mode when it actually shouldn't.
Fix a critical bug on God Helper that forcing both side chars guarding & disturbing AI when not in God Mode.
"Nearly God" Mode enhanced, now is "Ture God" Mode. (Technology of Wild Pointer online)
Striker small balance changes.

Add a new function to Striker Cooldown Switch, allow you to disable strikers under Nightmare Mode:

[State -2, Inner Mode Striker "respective" cooldown (unit:tick, 60ticks=1second)]
type = VarSet
trigger1 = 1
var(40) = 180   ;<----set to -1 will disable strikers in Nightmare Mode, but no influence on Inner Mode
;--------------
;New function:
;set to -1 to disable nightmare mode strikers
;Note: does NOT iinfluence inner mode (thus inner mode cooldown will be 180 by default)
-----------------------------------------------------------------
修复一个致命BUG，通过按键跳过开场时可能导致任何模式下的人物立即凶恶化。

修复一个致命BUG，混线检测helper不正确的干扰了普通模式对战，强制对手进入了防御状态，同时干扰了双方AI。

里模式援护变化：
夏米尔的暗黑雷光拳，出招时免疫飞行道具。注意蓄力时不免疫。
七枷社的大地冲击，现在不仅落地自动转身，而且蓄力完成瞬间也自动转身，确保向敌人所在的方向攻击。

在援护冷却时间的基础上增加一个附加功能：
[State -2, Inner Mode Striker "respective" cooldown (unit:tick, 60ticks=1second)]
type = VarSet
trigger1 = 1
var(40) = 180   ;<----通过将其设为-1，可以关闭梦魇模式下的援护系统。
注意：仅对梦魇模式有效，里模式不受影响（否则缺少援护的里模式与表模式无异）
设为-1后，里模式将使用默认的援护冷却时间（180帧）
-----------------------------------------------------------------
凶恶模式代码规范化：

启动方式与之前版本一致：
一败加强开关开启+梦魇模式一败=凶恶化。

凶恶化攻击面大幅提升，自准神级上升为神级。

攻击面：神（除对策外：高等精度混线，中等精度OTHK，超即死，变数弄；OTHK无变量记忆，无直死，无%N）。
