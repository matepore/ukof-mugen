  仿KOFZ，增加02UM隐藏超杀“天上地下唯我独尊”。
  仿KOFZ，增加空中神指。
  增加原创超必投“高尼兹合击”
  增加98UM物理攻击反弹技“神指・叛镜”

  反弹盾取消轻脚可形成97原作无限连。
  由于原作性能，关闭“cancel到超杀时的伤害递减修正”，因此超杀连段伤害大幅度增加。


能量修正参数：Orochi-EV-2.cns下
[State -2, poweradd adjustment multiplier]
type = VarSet
trigger1 = 1
fvar(10) = 1.0    《-----改变百分比，1.0=100%
;poweradd adjustment multiplier (positive float is acceptable)
;suggestions:
;set to 0.25 - 0.3 if you want Orochi to gain power like a common character.
;set to 1.0 if you want Orochi to gain power as a mighty BOSS.


伤害修正参数：Orochi-EV-2.cns下
[State -2, global damage adjustment multiplier]
type = VarSet
trigger1 = 1
fvar(3) = 1.0    《-----改变百分比，1.0=100%
;global damage adjustment multiplier (positive float is acceptable)
;suggestions:
;set lower than 1.0 if you want Orochi to do damage like a common character.
;set to 1.0 if you want Orochi to do damage as a mighty BOSS.
;set higher than 1.0 if ... well, become a cheap char.


不建议修改AI等级，精准连续技AI系统受AI等级影响复杂，降低AI等级可能人物傻瓜化，或毫无难度降低效果。


站立姿势有两个形式：仿KOFM盘腿神蛟姿势、原作的KOF97原版姿势。姿势可选择，请以记事本打开States/Orochi-EV-2.cns，查看如下代码
注意：KOFM姿势并未进行色表处理（过繁琐），非12号色表下可能颜色失真，不推荐使用。
[State -2, Stand Anim Select]
type = VarSet
trigger1 = 1
var(18) = 0
;Stand Anim Select :
站立动画选择
; 0 --- Catalyst Stand Anim
设置为0，将使用Catalyst灾难浩劫姿势（原作的KOF97原版姿势）
; 1 --- KOFM Stand Anim with background anim from Mizuchi
设置为1，将使用附带蛟的背景的KOFM姿势（仿KOFM盘腿神蛟姿势）
