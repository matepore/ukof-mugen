【Win版専用】
停滞を繰り返してましたが、98UM発売を機に一気に完成させました。

基本的に98UMベースの性能ですが
個人的に思い入れの強いキャラなので弄ってる箇所もあります。
具体的に列挙すると

・空中吹っ飛ばしが98準拠のフレーム数に（UMは発生が遅くなってます）
・ヘブンズゲイトの演出が96MAX版風に。
・技ボイスの一部（大半？）が96仕様に。
・バイス＆ルガールがストライカーに。

…ホントはパワー溜めとかも96画像にする予定だったんですが
かなり違和感が有ったので98のものにしています。
96マチュアは良いです。特にデスロウのCVとか。2002なんて無かった…

システムの仕様は基本的にGoenitz_Umと同様です。


========================【技一覧】===========================

方向キーの表記はキーボードのテンキーの配列と同様です。
システムの説明はWebページ等で98UMのものを参照して下さい。

一部の技はクイック版やガードキャンセル版が存在します。

・特殊動作

　【共通】
　挑発　…　S
　MAX発動 … XYA / ZC
　吹っ飛ばし攻撃　…　YB / C（空中可）

　【ADV専用】
　緊急回避・前　…　6+XA / 6+Z
　緊急回避・後　…　XA / Z

　【EX専用】
　攻撃避け … XA/Z
　パワーチャージ　…　XYA 押しっぱなし

・特殊技
クリマトリー … 6+A
バイス呼び出し（メイヘム）　…　YA / 6+YA / 6+S
バイス呼び出し（ブラックンド省略版）　…　4+YA / 4+S
ルガール呼び出し（ダークバリヤー）　… 2+YA / 2+S

・投げ技
デスブロウ　…　（相手の近くで）6+Y
バックラッシュ　… （相手の近くで）6+B

・必殺技
デスロウ　…　214+XorY（3回まで連続入力可）
メタルマサカー　…　214+AorB
ディーサイド　…　41236+AorB(Aのみ追撃可）
サクラリッジ　…　623+XorY
エボニーティアーズ … 2363214+XorY

・超必殺技
ヘブンズゲイト　…　2141236+AorB
ノクタールライツ　…　236236+XorY

・MAX超必殺技
ヘブンズゲイト　…　2141236+AorB / 2141236+AB
ノクタールライツ　…　236236+XorY / 236236+XY
