1.5 更新履历：
  增加变量fvar(4)作为蔚蓝/原版色调选择开关，不喜爱新版蔚蓝闪电色调的，请修改值为1，恢复为原版圣光色调。
  注意以下技能不受色调开关影响，因为它们在原作中就是那样的颜色：白罗灭精（恒为KOF2001圣光色调）、蔚蓝冲击（恒为KOFM蔚蓝色调）、展现合理的宇宙（恒为KOF2002UM圣光色调）。
  梦魇模式召唤援护时，本体完全无敌，不受投技影响。
  删除对DEF人物名称和DEF作者名称的trigger加密字段。
  但请注意尽量不要改变DEF人物名称和DEF作者名称，否则当双打模式为两个Orochi-EV，或者Orochi-EV和Igniz-EV时，若为里模式、浩劫梦魇模式，将造成援护冷却图标和浩劫梦魇特别技能条的错位。
  虽然强烈不建议修改DEF人物名称和DEF作者名称，但是游戏中的显示名称可以修改，它们并非同一个东西。

1.4 更新履历：
  增加02UM隐藏超杀“展现合理的宇宙”。
  增加变量fvar(3)作为全局伤害调整参数。注意请使用此参数修改全局伤害百分比，而不要使用CNS下的attack 参数，因为后者的修正无法作用于绝大部分超杀。
  超杀伤害上调约30%，其余伤害不变。
  修复一些里模式和梦魇模式的BUG。
  由于原作性能，关闭“cancel到超杀时的伤害递减修正”，因此超杀连段伤害大幅度增加。


蔚蓝/原版色调选择开关：Igniz-EV-2.cns下
[State -2, color design selector]
type = VarSet
trigger1 = 1
fvar(4) = 0      《-----0为蔚蓝闪电色调，1为原版圣光色调
;color design selector (1 or 0 is acceptable)
;set to 0 , azure blue lightning color design
;set to 1 , oroginal bright light color design


能量修正参数：Igniz-EV-2.cns下
[State -2, Poweradd global adjustment (multiplier)]
type = VarSet
trigger1 = !IsHelper(97864)
fvar(10) = 1.0   《-----改变百分比，1.0=100%
;poweradd global adjustment (multiplier) (positive float is acceptable)
;suggestions:
;set to 1.0 if you want igniz to gain power very closely to KOF2001/KOF2002UM (My suggestion)
;set to lower if you want igniz to gain less power like a common character.
;set to higher if you want igniz to gain much more power as a mighty BOSS.(nightmare...)


伤害修正参数：Igniz-EV-2.cns下
[State -2, global damage adjustment multiplier]
type = VarSet
trigger1 = 1
fvar(3) = 1.0    《-----改变百分比，1.0=100%
;global damage adjustment multiplier (positive float is acceptable)
;suggestions:
;set lower than 1.0 if you want igniz to do damage like a common character.
;set to 1.0 if you want igniz to do damage as a mighty BOSS.
;set higher than 1.0 if ... well, become a cheap char.
