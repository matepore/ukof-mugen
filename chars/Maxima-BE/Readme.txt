説明

　　　・ダブルボンバーの二段目と、ベイパーキャノンフェイントはスーパーキャンセル対応技です。
　　　　（スーパーキャンセルとは必殺技から超必殺技へのキャンセルの事です。）
　　　　またスーパーキャンセルには通常のゲージ消費に加え、デフォルト状態では、スキルゲージが１
　　　　設定変更時には、パワーゲージが５００必要です。

      ・ＭＸ−IIｂ　ファイナルキャノンには特殊追撃判定が設けてあり、
　　　　通常は追い討ち不可能な様々な場面で追い討ち出来ます。

　　　・ガードキャンセルふっ飛ばし攻撃でＫＯすることはできません。

　　　・Configにより、設定の変更が出来ます。

備考
　　　
　　　未搭載の技は技表にありません。

　　　原作と性能が異なる技がありますが、その様な仕様です。
　　　性能を変化させる理由は、
　　　１、原作を忠実に再現しようとすると、原作では全キャラに対して確定のコンボがMUGENでは確定しない事があり、
　　　　　それをなるべく防ぐため。
　　　２、MUGENのキャラは原作より性能が強めの傾向にあるので、単純な性能強化のため。
　　　３、私の技量不足、又は怠慢。
　　　以上です。
